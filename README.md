# LEADERBOARD

> A board that lets you get and send information to and from an API.

## Built With

- HTML
- CSS
- Javascript
- Webpack
- gitflow

## Live Demo

[netlify](https://artur-leaderboard.netlify.app/)

## Install

Clone this repository into your machine:
https://github.com/Arturgouveia1970/leaderboard.git
npm . init -y
npm start
npm install
